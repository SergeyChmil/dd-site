Создание нового поста. Нужно ввести заголовок, текст и загрузить фотографии. Если заголовок не указан, то пост автоматически опубликуется с заголовком "Без названия". Ни одно поле не является обязательным для заполнения, но хоть что-то нужно ввести. Можно загружать только снимки, можно только текст или вообще только заголовок. Можно комбинировать.
